# API
## Todo-list

### Общая структура задачи

| Параметр  | Тип | Описание |
| ------------- | ------------- |-|
| id  | UUID  | ID задачи |
| name  | string(20)  | Название задачи |
| description  | string(255)  | Описание задачи |
| deadline | Дата по ISO8601 | Срок выполнения |
| createdAt | Дата по ISO8601 | Дата постановки  задачи |
| status | waiting/in_progress/completed | Статус задачи |
| importance | number | Важность задачи (1 - 3) |

Пример структуры:
```json
{
    "id": "... uuid ...",
    "name": "Сделать сервис todo",
    "description": "Необходимо сделать сервис управления своими задачами",
    "deadline": "2025-07-15T09:01:07.815Z",
    "createdAt": "2025-07-15T09:01:07.815Z",
    "status": "waiting",
    "importance": 3,
}
```

### Ошибки

`2xx` - Всё окей

`400` - Неправильные данные отправили (или не тот формат)

`401` - Пользователь не найден / не авторизован

`404` - Задача не найдена

`500` - Сервер послал

### Получение всех задач пользователя

`GET /todo/all` 

Возвращает список всех задач пользователя.

*Параметры запроса*:
- **Без параметров**

*Параметры ответа*:
- Возвращает массив структур:
```ts
{
    "id": string,
    "name": string,
    "description": string,
    "deadline": string (Дата по ISO8601),
    "createdAt": string (Дата по ISO8601),
    "status": "waiting" | "in_progress" | "completed",
    "importance": number,
}
```

### Добавление задачи

`POST /todo/create`

Добавляет новую задачу.

*Параметры запроса*:
- Передаётся в формате JSON данные задачи:
```ts
{
    "name": string,
    "description": string,
    "deadline": string (Дата по ISO8601),
    "createdAt": string (Дата по ISO8601),
    "status": "waiting" | "in_progress" | "completed",
    "importance": number,
}
```

*Параметры ответа:*
- Возвращается id созданной задачи

### Изменение данных задачи

`PUT /todo/[taskId]/edit`

Изменяет существующую задачу по её ID.

*Параметры запроса*:
- Передаётся в формате JSON новые данные задачи:
```ts
{
    "name": string,
    "description": string,
    "deadline": string (Дата по ISO8601),
    "createdAt": string (Дата по ISO8601),
    "importance": number,
}
```

*Параметры ответа:*
- **Без параметров**


### Изменение состояния задачи (её статуса)

`PATCH /todo/[taskId]/edit?status=...`

Меняет статус задачи по её ID.

*Параметры запроса*:
- ID задачи (taskId) и её новый статус (status)

*Параметры ответа:*
- **Без параметров**

### Удаление задачи

`DELETE /todo/[taskId]`

Удаляет задачу по её ID.

*Параметры запроса*:
- ID задачи для удаления

*Параметры ответа:*
- **Без параметров**
